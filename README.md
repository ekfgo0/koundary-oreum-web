# Koundary (Frontend)

한국에서 유학중인 외국인 유학생들을 위한 커뮤니티 서비스

---

## 1. Project Overview (프로젝트 개요)

Koundary는 한 커뮤니티 내에서 같은 학교, 같은 나라 출신의 학생들과 소통하고 싶은 외국인 유학생들을 위한 커뮤니티입니다.


단순한 커뮤니티를 넘어서 같은 국적, 같은 학교 사람들만 모아서 소통 할 수 있으며 정보게시판을 통해 유학생들에게 유용한 정보들을 공유할 수 있도록 돕습니다.


다양한 유학생들과 소통하며 같은 관심사를 가진 친구들을 찾아보세요!

++ 여러 국적의 학생들을 위한 커뮤니티인만큼, 게시판에 언어가 다른 글이 올라왔을 때 본인의 국적(회원가입 시 선택한 국적)에서 사용하는 언어로 자동번역 되어 글이 보여지는 번역 기능이 추가되었습니다! (현재는 영어로만 가능)

## 3. Key Features (주요 기능)

### 로그인 / 회원가입
- 국적과 소속대학을 선택한 후, 각 대학에서 발급한 이메일 통해 이메일 인증하여 회원가입을 진행 할 수 있습니다.

### 메인 게시판
- 소속국가 : 같은 국적의 사람들과 소통할 수 있는 게시판입니다. (다른 국적의 사람들은 해당 게시판의 글을 열람할 수 없습니다.)
- 소속학교 : 같은 학교의 사람들과 소통할 수 있는 게시판입니다. (다른 학교의 사람들은 해당 게시판의 글을 열람할 수 없습니다.)
- 자유게시판: Koundary에 가입된 모든 회원이 함께 소통할 수 있는 게시판입니다.
- 중고거래 게시판: 유학생들에게 필요한 물품들을 거래 할 수 있는 게시판입니다.
- 정보게시판: 각 게시판에서 작성된 글들 중 '정보 글'이라고 선택된 글들을 모아 볼 수 있는 게시판입니다. (정보게시판에서도 글을 작성할 수 있습니다.)

### 글 작성
- 제목과 내용을 작성할 수 있으며 5MB 이하 이미지 첨부 가능합니다.
- 중고거래 게시판과 정보게시판을 제외한 게시판에서 글을 쓸 때에는 '정보 글' 선택 칸이 있으며, 선택 시 정보게시판에도 이중 업로드 됩니다.

### 내 프로필
- 프로필 이미지 변경 및 삭제
- 계정 정보 (아이디 확인, 비밀번호 수정, 회원탈퇴)
- 나의 활동 (내가 쓴 글, 댓글 단 글, 스크랩한 글)

### 신고
- 게시글 우측 상단에 있는 신고 버튼을 눌러, 신고 사유를 적어 관리자에게 전송할 수 있습니다.
